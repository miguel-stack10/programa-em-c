# 🛒 Sistema de Vendas em C

📌 Projeto desenvolvido em linguagem C simulando um sistema de vendas de e-commerce via terminal.

---

## 🚀 Funcionalidades

- ✔️ Validação de idade (maiores de 18 anos)
- ✔️ Aplicação de desconto por valor da compra
- ✔️ Desconto adicional para pagamento via Pix
- ✔️ Taxa para pagamento via boleto
- ✔️ Opção de parcelamento no cartão (até 6x)
- ✔️ Resumo final da compra

 🛠️ Tecnologias utilizadas

- Linguagem C
- Execução via terminal
